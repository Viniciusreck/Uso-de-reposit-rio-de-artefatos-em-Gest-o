# Vinny eye - Configuration Management Guide

## Overview
Vinny eye is a configuration management and artifact control system designed for software development teams.

## Features
- Centralized artifact storage
- Version control integration
- Team collaboration capabilities
- Configuration management

## Workflow
1. Clone the repository: `git clone https://github.com/Viniciusreck/Uso-de-reposit-rio-de-artefatos-em-Gest-o.git`
2. Make changes to artifacts locally
3. Commit changes: `git add . && git commit -m "message"`
4. Push to remote: `git push origin main`
5. Pull latest changes: `git pull origin main`

## Artifacts Location
All artifacts are stored in the repository root and can be retrieved using: `git clone` or `git pull`
